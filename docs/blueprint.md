# **App Name**: VideoVerse Download

## Core Features:

- Platform Selection: Allows users to select from a range of supported platforms (Instagram, Facebook, TikTok, YouTube).
- URL Input: Provides a user-friendly input field for pasting video URLs from selected platforms.
- Video Preview: Displays a preview of the video before downloading.
- Video Download: Enables downloading videos in the best available quality (no generative AI features).
- Proxy Integration: Integrates with a PHP proxy to bypass CORS restrictions when fetching video data.

## Style Guidelines:

- Primary color: Vibrant pink (#E91E63), reflecting the energy of multimedia content.
- Background color: Dark gray (#333333), for contrast and modern feel.
- Accent color: Electric blue (#03A9F4) for interactive elements.
- Body and headline font: 'Space Grotesk' sans-serif font for a modern, tech-oriented feel.
- Font Awesome icons for a consistent and scalable visual language.
- Clean, card-based layout for platform selection and video display.
- Subtle transitions and animations to enhance user interaction.
